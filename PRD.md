# 말씀카드 메이커 — PRD (제품 요구사항 문서)

## 1. 개요
말씀(성구)·시·명언 같은 글귀와 사진을 합쳐 한 장의 "말씀카드" 이미지를 만들어
저장·공유하는 웹앱. HTML 파일 하나로 동작하며 PC·휴대폰 브라우저에서 그대로 사용.

- 사용 빈도: 하루 1~5장 (소량)
- 비용: **완전 무료가 기본**. 유료 옵션은 선택 사항.
- 설치 불필요: `index.html`을 열기만 하면 됨 (오프라인에서도 합성·저장 가능, 단 AI/검색은 인터넷 필요)

## 2. 사용자 & 목표
- 교회/모임/개인이 현장에서 빠르게 말씀카드를 만들어 SNS·인쇄·배경화면으로 사용.
- 비전문가도 3단계(글 입력 → 배경 선택 → 저장) 안에 완성.

## 3. 핵심 기능

### 3.1 글귀 입력
- 본문(말씀/시/명언) 여러 줄 입력
- 출처/저자(예: "마태복음 11:28", "윤동주") 입력
- 자동 줄바꿈(한글 포함), 글자 크기 자동 맞춤 + 수동 조절

### 3.2 배경 이미지 (4가지 모두 제공, 현장에서 선택)
1. **무료 AI 그림 (Pollinations.ai)** — 가입·API키·비용 없음. 글귀/분위기로 배경 생성.
   - 그림체/사진체(실사) 분위기 프리셋, 프롬프트 수동 편집, "다시 생성"으로 재추첨
2. **내 사진 업로드** — 휴대폰 카메라/갤러리, PC 파일에서 직접 업로드
3. **무료 사진 검색 (위키미디어 커먼즈)** — API 키 불필요, 다운로드 안전(CORS 허용).
   키워드로 실제 사진 검색, "다음 사진"으로 순환
4. **DALL·E / 나노바나나(Gemini) 고급 옵션** — 사용자가 본인 API 키 입력 시에만 동작.
   더 높은 품질이 필요할 때 선택적으로 사용 (키는 브라우저 로컬에만 저장)

### 3.3 디자인 / 편집
- 스타일 프리셋: **정통(명조·은은한 하단 그라데이션)** / 모던(고딕·굵게·전체 음영) / 카드틀(여백 액자)
- 비율: 1:1(인스타) / 4:5(피드) / 9:16(스토리·배경화면) / A4 세로(인쇄)
- 한글 글꼴 선택: 명조·고딕·손글씨·굵은 제목체 (Google Fonts)
- 글자색, 정렬(가운데/좌), 세로 위치, 글자 외곽 그림자(가독성)
- 배경 어둡게/밝게 오버레이 + 투명도, 블러

### 3.4 출력
- **PNG 다운로드** (고해상도, 비율별 1080px 기준)
- **공유**: 모바일 Web Share API 지원 시 이미지 바로 공유
- 마지막 설정·API 키 자동 저장(localStorage), 초기화 버튼

## 4. 비기능 요구
- 모바일 우선 반응형. 한 화면에서 입력↔미리보기 즉시 반영.
- 외부 빌드/서버 불필요(정적 HTML 1개). 의존성은 CDN 폰트뿐.
- 다운로드 안전성: 캔버스 오염(CORS) 시 안내 메시지로 graceful 처리.
- 한국어 UI.

## 5. 디자인 방향 (Stitch/Figma 관련)
Stitch·Figma는 화면 시안을 그리는 도구이고, 실제 결과물은 동작하는 HTML이므로
동일한 수준의 정돈된 UI를 코드로 직접 구현: 차분한 색, 카드형 레이아웃,
넉넉한 여백, 좌(편집)·우(미리보기) 2단 → 모바일에서 1단으로 자동 전환.

## 6. 범위 밖 (이번 버전 제외)
- 로그인/계정, 서버 저장, 결제, 협업, 템플릿 마켓.

- [ ] 위키미디어 키워드 검색 사진으로 카드화 → 저장 가능 (CORS 안전)
- [ ] (선택) OpenAI/Gemini 키 입력 시 해당 엔진으로 생성 가능
- [ ] 4가지 비율, 3가지 스타일, 글꼴/색/위치 조절 동작
- [ ] 휴대폰 브라우저에서 동일하게 동작 및 공유
