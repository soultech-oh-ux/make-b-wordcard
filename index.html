잔잔한 자연 빛 · 하늘 수채화 사진 느낌(실사) 미니멀 따뜻한 톤

가입·API키·비용 없이 바로 생성됩니다 (Pollinations.ai). 글자는 그림에 넣지 않고, 카드에서 합성됩니다.

휴대폰에서는 촬영 또는 갤러리에서 선택할 수 있습니다. 업로드 사진은 항상 저장이 잘 됩니다.

위키미디어 + Openverse(플리커·박물관 등) 무료 사진을 함께 검색합니다. 키 불필요·다운로드 안전. 결과가 적으면 ‘고화질 추천’으로 예쁜 풍경 사진을 받아보세요.

⚠️ 나노바나나(Gemini) 권장 — 브라우저에서 바로 동작합니다. DALL·E(OpenAI)는 브라우저 직접 호출을 OpenAI가 차단하므로, 파일을 그냥 열어 쓰면 키가 맞아도 실패합니다(웹서버 배포 시에만 가능). API 키는 이 브라우저에만 저장됩니다 — 공용 PC에서는 사용 후 "키 지우기"를 눌러주세요.

키 발급 — 나노바나나: aistudio.google.com → "Get API key" (무료). DALL·E: platform.openai.com → API keys(결제 등록 필요, 브라우저 직접 호출 불가). 키 입력 후 "고급 엔진으로 생성"을 누르세요. 실패 시 화면 하단에 정확한 원인이 표시됩니다.
